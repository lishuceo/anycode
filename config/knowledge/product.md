# TapTap Maker (TapTap 制造)

## Overview

- Site: maker.taptap.cn
- Tagline: "想到就能做到" — zero-barrier AI game creation agent
- Launched: 2026-01-30
- Users describe game ideas in natural language; AI generates playable demos in minutes
- One-click publish to TapTap platform with built-in TDS services (login, multiplayer, cloud save, leaderboards)

## Tech Stack

- **AI Agent** — AI-driven game creation (understands natural language → generates game)
- **UrhoX** — AI-native game engine (web-based, cross-platform)
- **TDS** — TapTap Developer Services integration

## TapTap

- Mobile game community + third-party store, founded 2016
- "No joint-operation, zero revenue share" model — content-driven growth
- 48M+ MAU globally (2024 H1), 170+ countries
- Operated by XD Inc. (心动, HK.2400)
